¿Cómo funciona?

  1. 1
    Exporta el inventario desde tu sistema como archivo xlsx o csv. El archivo debe incluir las hojas de datos y la tabla de referencia de curvas.
  2. 2
    Sube el archivo aquí. El sistema procesará automáticamente el inventario y calculará la curva para cada producto.
  3. 3
    Escanea los LPN con el escáner Zebra. El sistema buscará automáticamente en el inventario cargado.
